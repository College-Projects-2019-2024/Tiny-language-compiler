# Overview
This is a simple compiler for the Tiny Language, a minimalistic programming language. The Tiny Language is designed for educational purposes and is easy to understand. This project is an Implementation of a scanner and a parser for the TINY programming language. The scanner takes the input file containing the source code written in Tiny language and produces tokens to be used in the second phase of the compiler, which is the parser.


# Features
* Lexical analysis (tokenization)
* Syntax analysis (parsing)
